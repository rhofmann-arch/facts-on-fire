⬡ Google Sheets Sync
📋 How Sheets Integration Works
Rosters are stored in a shared Google Sheet — one tab per teacher. All teachers and the coach see live data. No login system required beyond Google OAuth.
Setup Steps (one-time, done by tech coordinator)
1
Go to console.cloud.google.com → create a project → enable the Google Sheets API and Google Drive API.
2
Under APIs & Services → Credentials, create an OAuth 2.0 Client ID (Web Application). Add your site's URL as an authorized origin.
3
Create a master Google Sheet. Copy its Sheet ID from the URL: docs.google.com/spreadsheets/d/[SHEET_ID]/edit
4
Paste your Client ID and Sheet ID into the fields below and click Save.
